# PASBECONA型 LP生成プロンプト

以下のプロンプトをAI（Claude, ChatGPTなど）に入力して使用してください。
`【】`内の項目を自分の商品・サービスに合わせて書き換えてから実行します。

---

## プロンプト本文

```
あなたはセールスコピーライティングの専門家です。
以下の商品・サービス情報をもとに、PASBECONA（パスビーコーナ）の法則に従ったランディングページ（LP）のコピーを作成してください。

---

### 商品・サービス情報

- 商品/サービス名：【例：オンライン英会話スクール「TalkMaster」】
- ターゲット（ペルソナ）：【例：30代の会社員。昇進に英語力が必要だが、忙しくて学習時間が取れない】
- 主な悩み・課題：【例：英会話スクールに通う時間がない、独学では続かない、TOEICスコアが伸び悩んでいる】
- 商品の特徴・強み：【例：1回15分のマンツーマンレッスン、24時間予約可能、AIによる弱点分析】
- 価格帯：【例：月額9,800円（初月無料）】
- 実績・権威性：【例：受講者数5万人突破、TOEIC平均150点アップ、NHKで紹介】
- 特典・オファー：【例：初月無料、オリジナル教材プレゼント、30日間全額返金保証】
- 限定要素：【例：先着100名限定価格、キャンペーンは今月末まで】
- CTA（行動喚起）：【例：無料体験レッスンに申し込む】

---

### LP構成ルール（PASBECONA の法則）

以下の9つのセクションを順番通りに構成し、それぞれの役割を果たすコピーを書いてください。

#### 1. P（Problem：問題提起）
- ターゲットが「これは自分のことだ」と感じる具体的な悩み・問題を提示する
- 抽象的ではなく、日常のシーンが思い浮かぶリアルな描写にする
- 疑問文やシーン描写で読者の注意を引く
- **出力：キャッチコピー（ファーストビュー見出し） + 問題提起の本文（3〜5文）**

#### 2. A（Affinity：親近感・共感）
- 読者と同じ悩みを持っていた経験や、同じ立場の人の声を紹介する
- 「あなたの気持ちはわかります」という姿勢で、敵ではなく味方であることを示す
- 自己開示やストーリーテリングを活用する
- **出力：共感パート本文（3〜5文）**

#### 3. S（Solution：解決策の提示）
- 悩みの根本原因を明らかにし、解決の方向性を示す
- まだ商品の売り込みはせず、純粋な「知見の共有」として提示する
- 「実は○○が原因で…」「解決のカギは○○にあります」のような展開
- **出力：解決策パート本文（3〜5文）**

#### 4. B（Benefit：理想の未来・利得）
- 問題が解決した後の理想の状態を、読者目線で具体的に描写する
- 機能（Feature）ではなく、得られる体験・感情・結果（Benefit）にフォーカスする
- 「想像してみてください」「あなたは○○になれます」のような表現を使う
- ブレットポイント形式で3〜5個のベネフィットを列挙する
- **出力：ベネフィット見出し + ブレットポイント3〜5個 + 補足文**

#### 5. E（Evidence：証拠・根拠）
- 信頼性を担保するエビデンスを提示する
- 顧客の声（体験談）、数値データ、受賞歴、メディア掲載、専門家の推薦など
- 具体的な数字や固有名詞を入れてリアリティを出す
- **出力：お客様の声（2〜3名分）+ 実績データ**

#### 6. C（Contents：商品・サービスの詳細）
- 商品/サービスの具体的な内容を説明する
- 特徴、機能、プラン内容、含まれるもの、他社との違い
- 表形式やリスト形式で分かりやすく整理する
- **出力：サービス内容の見出し + 詳細説明 + 比較表（該当する場合）**

#### 7. O（Offer：提案）
- 価格の提示と、それが「お得である」と感じさせる演出をする
- 特典、ボーナス、保証（返金保証など）を含める
- 「○○円相当の特典が無料」「リスクゼロでお試しいただけます」などの表現
- **出力：価格提示 + 特典リスト + 保証内容**

#### 8. N（Narrow：絞り込み・限定）
- 期間限定、数量限定、対象者限定などの要素で緊急性・希少性を演出する
- 「今すぐ行動する理由」を読者に与える
- 煽りすぎず、合理的な理由を添えて限定性を説明する
- **出力：限定要素の提示（2〜3文）**

#### 9. A（Action：行動喚起）
- 具体的な次のステップを明確に示す
- CTAボタンのテキストを提案する
- 最後のひと押しとなるメッセージ（背中を押す一言）を添える
- 申し込みの簡便さを伝える（「たった30秒で完了」など）
- **出力：クロージングメッセージ + CTAボタンテキスト + 補足（申し込み手順など）**

---

### 出力フォーマット

- 各セクションに見出し（## 1. Problem / ## 2. Affinity ...）をつけて区切る
- コピーのトーンは「信頼感がありつつも親しみやすい」文体にする
- 読者に語りかける「あなた」視点を基本とする
- 1文は40文字以内を目安に、読みやすく区切る
- 各セクションの冒頭に【このセクションの目的】を1行で記載する

---

上記のルールに従い、LP全文を生成してください。
```

---

## 使い方

1. **`商品・サービス情報`の【】内を書き換える**
2. **プロンプト全文をAIに入力する**
3. **生成されたLPコピーをレビュー・調整する**

### カスタマイズのヒント

| やりたいこと | プロンプトへの追記例 |
|---|---|
| BtoB向けにしたい | 「トーンはビジネスフォーマルにしてください」 |
| 短めのLPにしたい | 「各セクション2〜3文に収めてください」 |
| HTML形式で欲しい | 「出力をHTML形式（Tailwind CSS使用）にしてください」 |
| 特定の業界向け | 「○○業界の専門用語を適切に使用してください」 |
| 感情に訴えたい | 「感情に強く訴えかけるエモーショナルなトーンにしてください」 |
